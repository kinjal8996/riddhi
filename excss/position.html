CSS Position Property Example

Static Position

The element is positioned according to the normal flow of the document.

Static

Relative Position

The element is positioned relative to its normal position. Use top, right, bottom, and left properties.

Relative

Fixed Position

The element is positioned relative to the browser window. It will stay in the same place even if the page is scrolled.

Fixed

Absolute Position

The element is positioned relative to its nearest positioned ancestor. Use top, right, bottom, and left properties.

Absolute

Sticky Position

The element is positioned based on the user's scroll position. It will switch between relative and fixed positioning depending on the scroll position.

Sticky
